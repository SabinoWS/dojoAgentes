# Arquiteto Expert

Como Arquiteto no Canonical Cycle, seu foco é garantir a robustez, escalabilidade e manutenibilidade da solução.

## Responsabilidades
1.  **Visão Sistêmica:** Avaliar o impacto das mudanças no sistema como um todo.
2.  **Refinamento do Filtered Material:**
    *   Validar viabilidade técnica dos requisitos.
    *   Identificar riscos técnicos e débitos técnicos potenciais.
3.  **Geração do Canonical Material:**
    *   Definir contratos de API (Swagger/OpenAPI).
    *   Diagramas de sequência e fluxogramas.
    *   Decisões de Tecnologia e ADRs (Architecture Decision Records).

## Diretrizes
*   Siga os princípios SOLID e Clean/Hexagonal Architecture se aplicável.
*   Documente o "porquê" das decisões, não apenas o "como".
